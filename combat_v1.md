# 戰鬥解算系統 v1（**確定性版**）

> 配合 `forces_v1.md`、`equipment_v1.md`、`logistics_v1.md`、`recon_v1.md`、`determinism_v1.md`。所有戰鬥結果由具體條件**確定**對應。**無擲骰、無機率**。
>
> ⚠️ 本檔中任何「X-Y%」範圍**僅作為歷史統計描述**，實際結算用 `determinism_v1.md` 公式得單一確定值。

---

## 0. 設計原則

戰鬥結果由 9 個確定量決定：
1. 兵力（人員 × 武器）
2. 組織度（cohesion）
3. 補給狀態（彈藥+燃料）
4. 經驗等級（veterancy）
5. 地形修正
6. 偽裝/能見狀態（誰看到誰、誰能瞄）
7. 兵種協同（combined arms）
8. 兵器對裝甲的物理穿甲（穿甲表）
9. 戰術狀態（突襲、預設、被動防禦、攻擊）

**不擲骰的不確定性來源：**
- 你不知道對方真實兵力（只能依偵察推估）
- 你不知道對方真實彈藥（不見得能維持火力）
- 你不知道對方真實組織度（疲勞、士氣，依過去戰鬥推測）

## I. 戰鬥類型分類

### 1. 偵察接觸（Skirmish）
- 規模：排-連級
- 時長：< 1 hour
- 結果：少量傷亡（< 1% 整師）
- 主要目的：情報蒐集、不是擊潰

### 2. 探查攻擊（Probing Attack）
- 規模：營級
- 時長：1-3 hour
- 歷史統計：傷亡 1-3%（依 `determinism_v1.md` §X 偵察接觸算式得確定值）

### 3. 預設陣地戰（Set-Piece Battle）
- 規模：團-師級
- 時長：3-12 hour
- 歷史統計：傷亡 5-15%（**這是歷史統計區間、實際數字由 Force Ratio 表確定**）

### 4. 攻擊與反擊（Attack / Counter-attack）
- 規模：師級
- 時長：4-24 hour
- 歷史統計：攻方 10-25%、守方 5-15%（**實際數字由 Force Ratio 表 + hour 累積確定**）

### 5. 城戰逐棟（Urban Grinder）
- 規模：營-團、緩慢
- 時長：可達數日
- 歷史統計：雙方各 20-40%（每 hour 結算累積到此規模）
- 詳見 `urban_combat_v1.md` + `determinism_v1.md` §V/§VI 公式

### 6. 追擊（Pursuit）
- 規模：對潰散部隊
- 時長：1-6 hour
- 歷史統計：被追擊方 30-60% 損失、追擊方 < 2%（每 hour 結算用 Force Ratio 表 × 追擊倍率 ×3）

## II. 戰鬥力計算公式（確定）

### 單位戰鬥力 = 8 個因子相乘

```
Combat Power (CP) = 
  Base_Power
  × (Strength% / 100)            ← 兵力百分比
  × (Org / 100)                  ← 組織度
  × Supply_Factor                ← 補給狀態（0.5-1.0）
  × Veterancy_Mult               ← 經驗倍率
  × Terrain_Mult                 ← 地形修正
  × Visibility_Mult              ← 能見狀態修正
  × Combined_Arms_Mult           ← 兵種協同
  × Tactical_State_Mult          ← 戰術狀態（突襲/預設/被動）
```

### Base Power 表（依兵種，師級基準）

| 兵種 | Base Power |
|------|------------|
| 美軍步兵師 | 100 |
| 美軍裝甲師 | 150 |
| 美軍機械化師 | 130 |
| 德軍步兵師 | 95 |
| 德軍 Panzergrenadier 師 | 130 |
| 德軍裝甲師 | 160 |
| 蘇軍步槍師 | 85 |
| 蘇軍裝甲師（坦克軍）| 145 |

### Strength% 與 Org / Supply

- Strength% = current_strength / TOE × 100
- Org / 100 = 直接百分比
- Supply_Factor：
  - 補給線完整、所有資源 > 60% = 1.0
  - 任一關鍵資源 < 30% = 0.7
  - 任一關鍵資源 < 10% = 0.4
  - 戰鬥所需資源耗盡 = 0.0（無法戰鬥）

## III. 戰鬥結算流程

### 步驟（每 hour 重算）

```
Hour N 結算流程：
1. 列出每個 hex 內參戰單位
2. 計算每個單位的 CP（戰鬥力）
3. 計算每邊 Total_CP（多單位合計）
4. 計算 Force Ratio = Attacker_CP / Defender_CP
5. 由 Force Ratio 對應結算表 → 雙方損失百分比
6. 計算具體傷亡（KIA / WIA / MIA / POW）
7. 計算組織度衝擊
8. 檢查潰散觸發
9. 若有戰車對戰車 → 加跑穿甲表結算
10. 寫入 state.json 與 combat_log
```

### Force Ratio → 損失對應表

| Force Ratio | 攻方 | 守方 | 地形變化 |
|-------------|------|------|---------|
| < 0.5 | -4% strength、-12% org | -0.5% str、-2% org | 攻方退（被迫）|
| 0.5 - 1.0 | -3% str、-10% org | -1% str、-4% org | 雙方原地 |
| 1.0 - 1.5 | -2% str、-7% org | -2% str、-7% org | 雙方原地 |
| 1.5 - 2.0 | -1.5% str、-5% org | -3.5% str、-12% org | 守方推 1 hex |
| 2.0 - 3.0 | -1% str、-3% org | -6% str、-20% org | 守方推 2 hex |
| 3.0 - 5.0 | -0.5% str、-2% org | -10% str、-30% org | 守方潰散風險 |
| > 5.0 | -0.5% str、-1% org | -15% str、-40% org | 守方必潰散 |

**注：** 這是 **1 hour 內**的損失。長時間戰鬥（如城戰）每 hour 重算、損失累積。
**★ 2026-06-05 史實校正**：strength 損失減半。原版 -4%/hr 連打 6hr = -24%（半天打殘一個師），
與史實「師級單日激戰傷亡 200-800 人(~1.5-5.7%)」差一個數量級。現值讓激戰一 tick(6hr)
均勢約 -12% strength，較貼近。**org（組織度）損失保留** —— 它代表疲勞/壓制/失序，
是可隨休整恢復的短期狀態，不是永久傷亡(見疲勞/重整補規則)。城戰另有加成不在此限。

### 傷亡分類（每個總損失百分比的內部分流）

對美軍/英軍 1944 標準（已優於德軍醫療水準）：
| 類型 | 比例 |
|------|------|
| KIA（陣亡）| 22% |
| WIA-輕傷（30 日內歸隊）| 38% |
| WIA-重傷（後送本國醫院）| 22% |
| MIA（失蹤）| 8% |
| POW（戰俘）| 10%（潰散時可達 40%）|

對德軍 1944：
| 類型 | 比例 |
|------|------|
| KIA | 28% |
| WIA-輕傷 | 32% |
| WIA-重傷 | 18% |
| MIA | 10% |
| POW | 12%（潰散時可達 50%）|

### 組織度衝擊計算

```
Org_Loss = Casualty_% × 1.5
  + Suppression_Penalty（連續戰鬥 6+ hour 每小時 -2）
  + Surprise_Penalty（被突襲方 -10）
  + Leadership_Loss_Penalty（指揮所被毀 -20、團長陣亡 -5）
  + Friendly_Fire_Penalty（誤擊 -10）
  - Resupply_Bonus（補給線完整 +1/hour 自然恢復）
  - Cover_Bonus（在城鎮/工事 -50% 衝擊）
```

### 潰散（Rout）觸發條件

**所有 3 條同時成立 → 該單位潰散：**
1. Org < 25
2. Casualty 24 hour 內 > 30%
3. 退路被切斷 OR 指揮鏈中斷

潰散後果：
- 該單位狀態 = ROUTED
- 強制撤退 2-3 hex 朝最近己方控制 hex
- 撤退途中可被追擊（追擊倍率 ×3 傷亡）
- 6 hour 後若補給線恢復 + 整補 = 恢復 Org 至 30（殘廢但可戰）
- 12 hour 不恢復 = 解散（殘部成為散兵）

### 投降（Surrender）觸發條件

**任一條件成立 + Org < 15：**
- 被四面包圍（無退路）
- 食物 < 5% AND 戰役> 7 天
- 整師指揮官陣亡 + 信號中斷
- 連續 48 hour 無補給

投降後果：
- 該單位狀態 = SURRENDERED
- 所有人員變 POW
- 裝備被己方繳獲（可分配）

### 穿甲判定（**確定**）

**規則：** 穿甲值 ≥ 裝甲有效厚度 = 貫穿 = 戰車被擊毀（**確定**）

#### 範例 1：Sherman 75 vs Panther（500m，正面）
- Sherman 75 穿甲：71mm
- Panther 正面裝甲：139mm
- **71 < 139 → 不穿、戰車安全**
- Sherman 75 不可能正面打贏 Panther（任何距離）

#### 範例 2：Sherman 76 vs Panther（500m，正面）
- Sherman 76 穿甲：99mm
- Panther 正面：139mm
- **99 < 139 → 不穿**
- Sherman 76 用 AP 仍不能正面破 Panther

#### 範例 3：Sherman 76 **HVAP** vs Panther（500m，正面）
- HVAP 穿甲：137mm
- Panther 正面：139mm
- **137 < 139 → 邊緣失敗**（差 2mm）
- HVAP 在 100m 內：162mm vs 139 → **PENETRATE**
- 所以 Sherman 76 用 HVAP 必須**接近到 100-200m** 才能正面破 Panther

#### 範例 4：Sherman 75 vs Panther **側面**（500m）
- Sherman 75：71mm
- Panther 側面：50mm
- **71 > 50 → PENETRATE**
- Sherman 必須**機動到 Panther 側面**才能殺它

#### 範例 5：Panther 75 L/70 vs Sherman 正面（任何距離 < 2000m）
- Panther 穿甲：138（500m）、102（2000m）
- Sherman 正面：63mm
- **任何距離都穿** → Sherman 任何情況下都被一發殺

#### 範例 6：Panzerfaust 步兵 vs Tiger I 側面（50m）
- Panzerfaust 60：200mm 貫穿（藥型裝藥）
- Tiger I 側面：80mm
- **200 > 80 → PENETRATE，Tiger 被殺**
- 但 Panzerfaust 射程僅 60m，必須步兵衝到極近

#### 範例 7：Bazooka vs Panther 正面（100m 內）
- Bazooka：100mm 貫穿
- Panther 正面：139mm
- **100 < 139 → FAIL**
- Bazooka 對 Panther 正面無效，必須側/後

#### 範例 8：Bazooka vs Pz IV 任何角度
- Bazooka：100mm
- Pz IV 正面：82mm、側 30mm、後 20mm
- **任何角度都穿、Pz IV 被殺**

#### 範例 9：CAS（P-47）vs 戰車**頂裝甲**（從天而降）
- 500lb 炸彈**直擊**頂裝甲（30mm）：可毀任何戰車——**但「直擊」機率極低**（火箭/炸彈對點狀戰車命中率<5%，見 §VII 校正）
- 5" HVAR 火箭：若命中可穿約 80mm，但命中率才是瓶頸
- **CAS 對戰車的真實價值是壓制/遲滯，不是擊殺**（ORS Mortain/Falaise 統計）；想真的殺戰車靠地面反裝甲

### 戰車對戰車戰鬥流程（1 hour 內）

```
條件：M4 連 × 17 輛 vs Panther 連 × 12 輛 在 500m 距離對峙

1. 雙方視距內、相互看到（依 recon_v1.md）

2. 每輛戰車每 hour 約 12-15 發 AP/HE 射擊（持續火力）

3. 命中數（**確定公式，見 `determinism_v1.md` §I**）：
   ```
   Hits = Rounds × Hit_Coefficient × Modifier
   ```
   **★ 2026-06-05 史實校正**：原係數 0.25-0.65（每發逾半命中）讓戰車戰 5 分鐘殲滅，
   嚴重高估。WWII 戰車對戰實際命中率：首發靜止對靜止約 30-50%，但**整場持續交戰**
   因煙塵、隱蔽、移動、緊張，**平均命中率僅 5-15%**（戰後 ORS 統計）。係數下修約 3-4 倍：

   Hit_Coefficient（每發命中機率，已校正）：
   - 雙方移動：0.06
   - 攻方移動 vs 守方靜止：0.10
   - 攻方靜止 vs 守方移動：0.13
   - 雙方靜止：0.15
   - 攻方 hull-down（守不是）：0.18
   - 雙方 hull-down 對峙：0.13
   （距離修正：≤300m ×1.3、500m ×1.0、800m ×0.7、1000m+ ×0.5）

4. M4 連射擊 = 17 輛 × 13 發 × 0.15（雙方靜止 500m）= **~33 發命中**（確定）
5. 每發命中對 Panther 進行穿甲檢查
   - 500m 正面 Sherman 75 = FAIL（0 殺）
   - 8 輛 76mm Sherman + HVAP（4 發/車）：500m 正面邊緣失敗、接近 200m 成功
   → 殺敵戰車數 = HVAP 命中數 × 穿甲成功率 × 接近距離係數

6. Panther 連射擊 12 × 13 × 0.13 = **~20 發命中**
   - 500m 正面任何 Sherman = 穿甲 SUCCESS
   - 殺傷率 0.8 → 約 16 輛被擊毀

   **持久戰，非瞬殺**：一個 hour 的交戰下來，Panther 命中 ~20 發、殺 ~16 輛 Sherman；
   Sherman 75 正面打不穿 Panther（須側擊/HVAP 近距）。但這是**一整個 hour** 的結果，
   不是 5 分鐘——戰車戰是反覆機動、找掩體、側翼包抄的持久過程。
   數量優勢與側擊機動因此有時間發揮（見戰術應對）。
```

### 戰術應對

**美軍對 Panther 應對：**
1. **絕不正面對抗**——側翼包抄
2. **76mm Sherman 集中**（兵力優勢）
3. **HVAP 接近到 200m 以下使用**
4. **CAS 從頂攻擊**（P-47 火箭+炸彈）
5. **數量優勢**——5 輛 Sherman 從側面圍 1 輛 Panther（一 hour 內 5 × 13 發 × 0.15 ≈ 10 發命中、穿甲 71mm vs Panther 側 50mm 全穿、**高機率擊殺**；側擊+數量是 Sherman 對抗 Panther 的正解）
6. **步兵 Bazooka 從側/後（100m 內）**——巷戰特別有效

**德軍 Panther 應對：**
1. **保持距離 > 1,500m**——讓 Sherman 即使是 76mm 也打不穿
2. **正面對敵**——側面是 50mm、脆弱
3. **接受戰術範圍限制**（不上城戰，因步兵 Bazooka 太危險）

## V. 步兵戰鬥（不是穿甲、是火力與壓制）

### 步兵單位的「火力等價」

每步兵連的火力等價（步槍兵基準 = 1.0）：
- M1 Garand 步槍兵 × 100 = 100 火力點
- BAR 自動步槍 × 1 = 4 火力點
- .30 LMG × 1 = 7 火力點（持續壓制）
- .50 HMG × 1 = 10 火力點（壓制+反裝甲輕車）
- 60mm 迫擊 × 1 = 6 火力點（曲射）
- 81mm 迫擊 × 1 = 12 火力點
- Bazooka × 1 = 視對手而定（vs 步兵 = 3 火力點，vs 戰車 = 一發殺機）

**美軍標準步兵連**（193 人）約 240-280 火力點。
**德軍步兵連**（180 人，MG42 加成）約 220-260 火力點。

### 步兵交戰結算

兩個步兵團對抗（每團 3 連 ≈ 800 火力點）：
- A 團 800 vs B 團 800
- 守方在城鎮 = ×1.8
- A 攻方在開闊地 = ×0.6
- 實際 480 vs 1,440 = 1:3 守方優勢
- 攻方無法突破（依 Force Ratio 表，0.3 ratio）

### 步兵清城戰術

詳細於 `urban_combat_v1.md`。簡述：
- 步兵分組（4-6 人小組）逐建築物推進
- 每建築物 30-45 分鐘清理
- 守方在地下室/暗道每棟 +0.5 倍率
- 工兵爆破門窗 = +1 倍率
- Bazooka/Panzerfaust 開門 = +0.5 倍率

### 砲擊類型

#### 1. 預設彈幕（Pre-planned Barrage）
- 砲兵 FDC 預先計算座標
- 命令下達 → **5 分鐘內**開火
- 精度高（誤差 ~50m）
- 用於：攻擊發起、預設殺戮區

#### 2. 即時呼叫（Call-for-Fire / TOT）
- FO 觀察到目標 → 用無線電/電話呼叫
- 程序：FO → FDC → 砲兵 → 開火
- 美軍 TOT（Time-on-Target）= 多個砲組同時抵達
- 時序：呼叫後 **3-7 分鐘**開火
- 精度：誤差 ~75m

#### 3. 修正射（Adjust Fire）
- 第一次彈著後 FO 報修正
- 每次修正後 1-2 分鐘再射
- 5-10 分鐘抵達精確命中
- 美軍 FO 技術較德軍精準

#### 4. 反砲擊（Counter-battery）
- DF + 聲測 + 觀察機定位敵砲位
- 反砲擊發起 10-30 分鐘
- 摧毀率：每砲彈藥 30 發 → 1 門敵砲被摧毀

#### 5. 干擾射擊（Interdiction）
- 對路口、橋樑、補給線間歇射擊
- 不期擊殺、僅遲滯敵運動

### 砲擊效果（步兵目標）— **見 `determinism_v1.md` §III 公式**

```
傷亡 = Rounds × Lethality × Exposure × Density
```

| 砲擊類型 | 公式輸出（1 hour、開闊散開營） |
|---------|--------------------------------|
| 6 門 81mm 迫擊持續射擊 1 個連 | 6×60×4×0.18×0.7×0.3 = **54 人傷亡**（連級密度）|
| 12 門 105mm 榴砲集中射擊 1 個營 | 12×60×2.5×0.30×0.7×1.0 = **378** ÷ 1hr 持續 → **平均約 38 人/小時** |
| 24 門 105mm 火力急襲（5 分鐘 540 發）| 540×0.30×0.7×1.0 = **113 人** |
| 12 門 155mm 重榴 30 分鐘 | 12×30×1.5×0.55×0.7×1.0 = **208 人** |
| 軍級 4.2"+105+155 全力急襲 30 分 | 12×30×4×0.30 + 24×30×4.5×0.30 + 12×30×1.5×0.55 = **1,431×0.7×1.0 = 1,001 人**（密集急襲、暴露目標） |

注：數字依目標暴露/密度具體調整。詳見 determinism_v1 §III。

### 砲擊效果（裝甲目標）— **見 `determinism_v1.md` §III**

```
殺戰車數 = Rounds × Tank_Kill_Coef × Tank_Exposure
```

Tank_Kill_Coef：
- 60mm 迫擊：0
- 81mm：0.001
- 105mm HE：0.005
- 155mm HE：0.02
- 155mm 直擊頂裝甲：0.2

Tank_Exposure：
- 開闊行軍：1.0
- hull-down + 偽裝：0.05
- 樹蔭內：0.01

範例：
- 直擊頂裝甲 = 必殺（任何戰車）
- 100 發 105mm vs 100m 內開闊戰車 = 100×0.005×1.0 = **0.5 輛/百發**（即「每 200 發殺 1 輛」）
- 戰車**hull-down + 距離 500m**：105mm 級 0.005×0.05 = **0.00025 殺機/發**（基本無效）
- **15cm 重榴**直擊 = 必殺

### 砲擊效果（工事/建築）

- 砲擊 + 建築 = 工事退化（保護倍率每 hour -0.2）
- 連續砲擊 4-6 hour → 建築變廢墟（守方倍率歸 1.0）
- 155mm 重砲 + 巷戰建築：每 hour 守方戰力減 8%（具體值，非範圍）

### 反砲擊（Counter-Battery）流程

```
1. 美軍砲擊 → 德軍 SCR-625 DF 測向到砲位
2. 軍級 PI + 觀察員確認 → 位置鎖定
3. 軍級反砲擊命令下達 → 5-10 分鐘
4. 反砲擊執行（30-100 發針對該砲位）

對抗策略：「打了就走 (shoot-and-scoot)」
- 美軍 M7 Priest 自走砲（裝甲師有）開火後立即機動 → 反砲擊失效
- 拖砲（leFH 18、105mm 拖砲）拖移時間 30-45 分 → 反砲擊可能擊毀
```

### 對特定戰車的 CAS（★ 2026-06-05 史實校正）

**原版「P-47 對開闊戰車必殺」嚴重高估。** 戰後 ORS（作戰研究組）對 Mortain/Falaise 的調查：
飛行員聲稱的戰車擊殺數遠超實際——火箭對點狀戰車目標**命中率僅 0.5-5%**，多數被算成
「擊毀」的戰車其實是乘員棄置。空中對戰車的**真正價值是壓制與遲滯，不是直接擊殺**。

- 1 個 P-47 sortie 對戰車單位（一群）：
  - **主要效果：壓制** —— 該 hour 機動停滯、組織度 -20、被迫散開找掩體
  - **擊殺：機率低** —— 開闊暴露 ≈ 每 sortie 擊毀 0-2 輛(命中率<5%)；hull-down/樹蔭更低
  - 樹蔭內：基本 MISS（看不到）
- **正確理解**：CAS 的戰場價值是「把德軍裝甲釘在原地、打斷其機動反擊」，
  而非「從天上點殺戰車」。要真的消滅戰車仍須地面反裝甲(側擊/HVAP/Bazooka)。
- 例外：**延遲信管 1000lb 直擊**仍可毀戰車(見下)，但那是運氣好的直接命中，非常態。

### AA 對 CAS 效果 — **見 `determinism_v1.md` §VII 公式**

```
P-47 損失 = Sum(AA × Rate) × Exposure_Time × Plane_Vulnerability
```

範例：17SS AA（20mm × 18 + 37mm × 9 + 88mm × 12）對 8 機 P-47 編隊（單次攻擊 3 分鐘暴露）：

```
AA_Power = (18×0.005 + 9×0.012 + 12×0.003) × 3 分鐘 = 0.702
擊落數 = 0.702 × 1.0（低空 P-47） = 0.702 → 取整 = 1 機損失（確定）
損失率：1/8 = 12.5%
```

美軍空優的決定性：即使損失 1 機，攻擊已完成、地面目標已重創。

## XI. 完整戰鬥範例

### 範例：Tick 5 (15,9) 戰鬥（Run 1 重算）

#### 條件
- 美軍 4ID 從 (10,10) 強渡中橋抵 (15,9)
- 德軍守方：17SS 擲彈兵連 (16,9) + 殘存 88 (15,9-10) + 75mm PaK 殘部

#### 戰鬥力計算（攻方 4ID）
- Base Power: 100（美軍步兵師）
- Strength: 83%
- Org: 46
- Supply Factor: 0.9（補給完整）
- Veterancy: ⭐⭐⭐ = 1.0
- Terrain: 過河強渡剛上岸 ×0.7
- Visibility: 雙方互見 = 1.0
- Combined Arms: 步+戰（CAS 包 C 同時打殺戮區）+ 砲（1ID 砲擊）= 三兵種 ×1.5
- Tactical: 強渡剛上岸 1 hour 內 ×0.7

**CP_attacker = 100 × 0.83 × 0.46 × 0.9 × 1.0 × 0.7 × 1.0 × 1.5 × 0.7 = 25.3**

#### 戰鬥力計算（守方）
- Base Power: 95（德軍步兵基準，這是擲彈兵連+砲位，近似團級）
- Strength: 設 70%（連+砲位 + 已半失能 AT 殺戮區）
- Org: 65
- Supply Factor: 0.85
- Veterancy: ⭐⭐⭐⭐ = 1.15
- Terrain: 城鎮邊緣防禦 ×1.8 + 戰壕 ×0.5 疊加 = 2.3
- Combined Arms: 步+砲 = 雙兵種 ×1.3
- Tactical: 預設殺戮區 + AT 火網 = ×1.2（特殊預設）

**CP_defender = 95 × 0.70 × 0.65 × 0.85 × 1.15 × 2.3 × 1.3 × 1.2 = 145.2**

#### Force Ratio
- 攻方 25.3 / 守方 145.2 = **0.17（< 0.5）**

#### 結果（依表）
- 攻方 -8% str、-12% org
- 守方 -1% str、-2% org
- 攻方退（被迫）

#### 套用到 4ID
- 4ID 兵 83% → 83 × 0.92 = 76%（**和 Run 1 我大概算 68% 接近但更精確**）
- 4ID 組 46 → 46 × 0.88 = 40
- 退回西岸或原地

#### 套用到 17SS 守方
- 連+砲位 70% → 69%
- 65 → 63

#### 但若 CAS 包 C 打掉一半 AT 殺戮區（如 Run 1 實際發生）
- 守方 Terrain ×2.3 改為 ×1.5（殺戮區半失能）
- 守方 CP = 95 × 0.7 × 0.65 × 0.85 × 1.15 × 1.5 × 1.3 × 1.2 = 94.5
- Force Ratio = 25.3 / 94.5 = 0.27（仍 < 0.5）
- 攻方仍會敗、但傷亡稍輕

**這就是 Run 1 我「感覺扣 4ID 100→83」的具體公式版本——確定可重複算**
