# 後勤系統 v1 — WWII 師級補給嚴謹模型

> 配合 `rules_v2.md` 時間系統使用。每 hour 結算消耗、resupply、interdiction。

---

## 0. 設計原則

- 每個師追蹤 **8-10 個資源欄位**（依兵種）
- 消耗依**活動等級**（5 級）× 兵種乘數
- 補給靠**車隊系統**（明面追蹤）
- 補給線斷 = 不補但仍消耗
- 倉庫深度有限——戰役級資源池

---

## 1. 資源類型（每師追蹤）

### 共用（每師都有）
| 欄位 | 代號 | 說明 |
|------|------|------|
| 燃料 | **POL** | 汽油+柴油+機油，0-100% |
| 輕兵器彈 | **SA** | 步槍/SMG/手槍/MG 通用 |
| HE 彈 | **HE** | 迫擊、榴彈、砲兵高爆 |
| AT 彈 | **AT** | 反戰車（Bazooka/Panzerfaust/PaK/坦克砲穿甲彈）|
| 口糧 | **RAT** | 食物+水（C/K 口糧、Wehrmacht ration）|
| 醫療 | **MED** | 繃帶、血漿、嗎啡 |
| 零件 | **PARTS** | 車輛零件、武器替換、引擎 |

### 專門（依兵種）
| 兵種 | 額外欄位 |
|------|---------|
| 砲兵 | **SMOKE**（煙幕彈）、**ILLUM**（照明彈）|
| 工兵 | **DEMO**（炸藥）、**BRIDGE**（架橋器材）、**MINES**（地雷）|
| 偵察 | **RECON**（偵察器材：望遠鏡、相機、無線電電池）|
| 防空 | **AA**（88mm AA 彈、20mm Flak）|
| 通信 | **CRYPTO**（密本、密碼機紙帶、無線電電池）|

### 德軍特殊類型（Panther/Tiger 師需另立）
| 武器 | 專用彈藥代號 |
|------|------------|
| Panther（Pz V）75mm L/70 | **AT-Pz5** ← 不能跟 Pz IV 75mm L/48 互換 |
| Tiger I 88mm L/56 | **AT-Tg1** |
| Tiger II 88mm L/71 | **AT-Tg2** |
| Pz IV 75mm L/48 | **AT-Pz4** ← 大部分德軍裝甲師主力 |
| Stug III/IV | **AT-St** |

**為什麼分**：1944 後期德軍 Panther 與 Tiger 的彈藥供應線優先序不同，常見「Tiger 部隊 Panther 彈藥」這種無解狀況。

## 2. 消耗等級（5 級 × 每 hour）

### 等級對照
| 等級 | 描述 |
|------|------|
| **L0 駐紮整補** | 不動、整補中 |
| **L1 行軍** | 機動但無接敵 |
| **L2 警戒/觀察** | 前哨、偵察、防禦待命 |
| **L3 戰鬥** | 接敵交火、開砲 |
| **L4 重戰鬥** | 全力突擊、城戰、強渡 |

### 步兵師每 hour 消耗（基準）

| 資源 | L0 | L1 | L2 | L3 | L4 |
|------|----|----|----|----|----|
| POL | 0.3% | 2.0% | 1.0% | 2.5% | 4.0% |
| SA | 0.1% | 0.2% | 0.5% | 2.0% | 4.0% |
| HE | 0% | 0% | 0.3% | 1.5% | 3.5% |
| AT | 0% | 0% | 0.2% | 1.5% | 4.0% |
| RAT | 0.7% | 0.7% | 0.7% | 0.9% | 1.0% |
| MED | 0.1% | 0.2% | 0.5% | 2.0% | 5.0% |
| PARTS | 0.2% | 0.5% | 0.3% | 1.0% | 2.0% |

> **★ 2026-06-05 油料校正（貼近 1944-08 史實）**：原 POL L1=1.0% 讓裝甲師可機動 33hr，與本戰役時空（1944-08 盟軍油荒、巴頓裝甲卡在塞納河）矛盾。已上修：裝甲師 L1 行軍 = 2.0%×3.0 = **6%/hr → ~16hr（約 2.7 tick）耗盡**，符合「一個基數油料約一日激烈機動耗盡」。POL <20% 移動 -50%、<10% 完全停（見危機表）。**裝甲突進必須顧油，不能無限機動。**

### 裝甲師乘數
- POL **×3.0**（油老虎）→ L1 行軍 6%/hr、L3 戰鬥 7.5%/hr、L4 12%/hr
- AT **×2.5**（坦克砲彈消耗高）
- PARTS **×2.0**（複雜機械）
- HE **×1.5**
- 其他同步兵

### 摩化/擲彈兵師乘數
- POL **×2.0**
- 其他同步兵

### 砲兵集團（軍/集團軍級附屬）乘數
- HE **×4.0**
- AT **×1.5**
- POL **×1.5**

## 3. 補給線與車隊

### 補給線追蹤
- 每師有一條**補給線**從師後方延伸到**集團軍/陣營邊緣**（補給總站）
- 路徑必須**連續通過己方控制 hex**——遇敵控制或無法通行 hex 即斷
- 河流必須**通過己方控制的橋**
- 補給線可分支（如同一橋為三師共用）

### 車隊系統
- **每 6 hour**（每 tick）每師發出 1-2 個補給車隊
- 車隊由**集團軍級**派出（不算在師戰力中）
- 車隊類型：
  | 類型 | 載運 | 速度 | 偵察被見性 |
  |------|------|------|---------|
  | **POL 車隊** | 燃料 50% / 1 趟 | 4 hex/hour | 高（油料車明顯）|
  | **彈藥車隊** | SA+HE+AT 共 40% / 1 趟 | 3 hex/hour | 中 |
  | **混合補給車隊** | POL+SA+HE+AT+RAT 各 **22%** / 1 趟 | 3 hex/hour | 中 |
  | **重武車隊** | 大型武器/坦克彈/工兵器材 | 2 hex/hour | 高 |

### 車隊抵達流程
```
集團軍補給站 → 車隊出發 → 沿補給線推進 → 抵達師後方補給點 → 卸載 → 補給該師
```

抵達後資源依**載運量**回補給該師（不超過 100%）。

### 戰役級資源池（戰略限制）
- 每方有**戰役總物資池**（雙方平均 100,000 噸/戰役）
- 每車隊出發消耗對應數量
- 池子用盡 = 無新車隊可派
- 玩家可選擇優先補哪類資源（POL / AMMO / 混合）

## 4. 補給線中斷的後果

### 中斷狀態
| 狀況 | 影響 |
|------|------|
| 補給線完整 | 每 tick 1-2 車隊正常抵達 |
| 補給線部分受阻（敵游擊隊干擾） | 車隊延遲依干擾規模（`determinism_v1.md` §IX 公式得確定值）|
| 補給線完全斷（橋斷/敵佔關鍵 hex） | 車隊**無法抵達**、該師消耗自有庫存 |
| 補給線斷>12 hour | POL 與 AT 庫存可能降至 30% 以下、進入危機狀態 |

### 危機閾值（任一資源 <X%）
| 資源 | 危機閾值 | 後果 |
|------|---------|------|
| POL | <20% | 裝甲師移動 -50%；<10% 完全停 |
| SA | <30% | 戰鬥強度 -30%；<10% 不可主動接敵 |
| HE | <30% | 砲擊強度 -30%；<10% 不可間瞄 |
| AT | <30% | 裝甲對抗 -40%；<10% 無法擊毀重戰車 |
| RAT | <30% | 組織度上限 -10；<10% 連 -20 |
| MED | <30% | 戰場傷亡組織度衝擊 +50% |
| PARTS | <30% | 車輛故障率 +30%；<10% 戰車組織度下降 |

## 5. Interdiction（補給線打擊）

### 干擾方式（從輕到重）
| 行動 | 影響 |
|------|------|
| 偵察接觸（你發現對方車隊路線）| 情報 only |
| 砲擊補給點 | 該 hour 車隊延遲、損失依砲擊強度（`determinism_v1.md` §III 砲擊公式）|
| 機械化襲擾隊（如 Run 1 90ID 機動小組）| 持續干擾期間 -30% 載運量 |
| 阻斷關鍵 hex（佔橋/關鍵路口） | 補給線改道、延遲 1 hour 額外 |
| **完全切斷**（殺光鄰近補給點+佔橋）| 完全斷供 |

### 范例（Run 1 重算）— **見 `determinism_v1.md` §IX 公式**

```
損失率 = Attacker_Strength × Defender_Vulnerability × Surprise_Factor / 100
```

Run 1 中 90ID 機動小組襲擾德軍東補給線。新公式下的精確影響：

- 90ID 派 1 個機動營（400 人，含 jeeps + M8 Greyhound）= Attacker_Strength **12.0**
- 部署位置 (22-24, 11-13) 阻擾路網、完全突襲 Surprise = **1.0**

**對 POL 車隊**（Vulnerability 1.5）：
```
損失率 = 12.0 × 1.5 × 1.0 / 100 = 0.18 = 18%
POL 車隊載運 50% → 18% × 50% = 該 tick POL 補給 -9%
```

**對彈藥車隊**（Vulnerability 1.3）：
```
損失率 = 12.0 × 1.3 × 1.0 / 100 = 15.6%
彈藥車隊載運 40% → 該 tick SA -6%、HE -6%、AT -6%
```

**對 17SS / 2Pz 整體**：該 tick POL -9%、HE -6%、AT -6%

差別：原本「-5 補給」變成精確的「POL -9%、HE -6%、AT -6%」→ 直接反映在攻城火力下降。

```json
"supply": 70
```

改為：
```json
"resources": {
  "POL": 80,
  "SA": 90,
  "HE": 85,
  "AT": 90,
  "RAT": 95,
  "MED": 85,
  "PARTS": 80
```

裝甲師加：
```json
"AT_subtype": "M3_75mm_Sherman"  // 影響 AT 彈藥可否分享
```

```json

  "convoys_pending": [],
  "interdicted_routes": []
```

## 9. 結算流程（每 hour 加上）

```
4.5 [新] 補給結算
  - 每師依當前活動等級扣資源
  - 檢查補給線是否完整
  - 抵達車隊 → 補給
  - 計算危機閾值狀態
  - 套用 critical 狀態效果（移動 ×0.5、戰鬥 -30% 等）
```

## 10. 範例：Run 3 Tick 1 結算（假設你照舊隱蔽 3 師）

```
Hour 1 (06:00-07:00) — 全軍 L0 駐紮整補 → 微量消耗
Hour 2 (07:00-08:00) — 90ID L1 行軍 → POL -1%, RAT -0.7%
Hour 3-6 — 3AD 機動 (5,5)F → (9,3)F (L1)
            → POL -1% × 4 hour × 3 (armor) = -12% POL
            
3AD 結束時 POL：80% → 68%
（裝甲師移動了 4 hex 燃料花掉 12%，這就是裝甲師「跑得遠就沒油」的真實）
```
