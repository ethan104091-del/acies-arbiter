# 天氣系統 v1（**確定性版**）

> 1944-08 法國月度天氣表。**雙方對稱可見**——天氣是公開資訊。每天天氣由戰役腳本**寫死**，不擲骰。

---

## 0. 設計原則

- **天氣表確定**：戰役期間每 6 hour 的天氣**預先寫定**
- **雙方對稱可見**：氣象觀測+目視，沒有資訊不對稱
- **戰役起始前可預知 12-24 hour**（軍級氣象組能力）
- **超過 24 hour 預測精度下降但仍有方向性**
- **影響函數確定**：雨天 CAS 不可飛、泥濘移動 ×0.5

---

## I. 天氣參數（5 維）

每 6 hour 戰況點記錄：

| 參數 | 範圍 | 例 |
|------|------|---|

| **溫度 Temp** | 寒 < 5°C / 涼 5-15 / 溫 15-25 / 暖 > 25 | 溫 18°C |

## II. 天氣對戰場的確定影響

### 對移動

| 條件 | 移動倍率 |
|------|---------|
| 晴朗、乾燥 | ×1.0 |
| 小雨持續 6+ hr → 道路濕滑 | ×0.85 |
| 中雨 6+ hr → 部分泥濘 | ×0.7 |
| 大雨 12+ hr → 全面泥濘 | ×0.5 |
| 暴雨 → 道路毀損、過水 | ×0.3 |
| 霧 → 視距僅 0.5 hex | 不影響速度但接敵風險高 |
| 強風 → 行軍正常但飛行高風險 | ×1.0（地面）/ 空中影響 |
| 寒冷 < 5°C 持續 → 機械故障率 +20% | ×0.9 |

**注：** 裝甲車輛在泥濘中受影響特大（履帶陷入）。Sherman/Panther 重型在大雨後 12 hr 移動限 ×0.4。

### 對航空

| 條件 | CAS sortie 可否 |
|------|--------------|
| 晴朗或多雲（雲底 > 1,500m）| ✅ 可飛、正常效率 |
| 陰天（雲底 800-1,500m）| ✅ 可飛、效率 ×0.85 |
| 厚雲低（雲底 < 800m）| ⚠️ 不可俯衝轟炸、僅可低空掃射、效率 ×0.5 |
| 雨中（中雨以上）| ❌ 通常取消、可堅持但精度差 ×0.4 |
| 霧 | ❌ **不可飛** |
| 強風 > 30 mph | ⚠️ 精度差 ×0.6 |
| 夜間 | ❌（A-20 夜入侵者除外，可飛但精度 ×0.4）|

### 對偵察

| 條件 | TacR | PR | L-5 | 地面偵察 |
|------|------|----|-----|---------|
| 晴朗 | 1.0 | 1.0 | 1.0 | 視距 5 hex |
| 多雲 | 0.95 | 0.8 | 0.95 | 5 hex |
| 陰天 | 0.8 | 0.5 | 0.8 | 4 hex |
| 厚雲低 | 0.5 | 0.2 | 0.5 | 3 hex |
| 中-大雨 | 0.3 | 0.1 | 不可飛 | 3 hex |
| 暴雨 | 不可飛 | 不可飛 | 不可飛 | 2 hex |
| 霧 | 不可飛 | 不可飛 | 不可飛 | **0.5-1 hex** |

**1.0 = 完整 recon_v1.md 觀察結果**，0.5 表示「能見的徵候只看到一半」。

### 對戰鬥

| 條件 | 對戰鬥 |
|------|--------|
| 晴朗 | 標準 |
| 雨/霧 | 步兵能見差、白刃戰/巷戰相對有利 |
| 大雨 | 步兵移動疲勞 +20%、火砲精度 ×0.7 |
| 暴雨/暴風 | 攻勢通常取消（攻方倍率 -30%）|
| 霧 | 突襲機率大增（雙方）、伏擊倍率 ×1.5 |
| 寒冷 | 火炮潤滑問題、戰車啟動慢、人員凍傷風險（短時間影響低）|

### 對補給

| 條件 | 補給線 |
|------|-------|
| 晴朗 | 1.0 |
| 中雨 | 0.85（卡車減速）|
| 大雨/泥濘 | 0.6 |
| 暴雨 | 0.3 |
| 道路毀損 | 0.2（直到工兵修復）|
| 橋樑被洪水沖毀（暴雨後）| 0（直到修復或繞道）|

## IV. 戰術影響的具體範例

### Tick 1（晴朗）的選擇空間
- **CAS 全力可用**：每天 8-12 個 sortie 配額
- **TacR/PR 順利**：偵察情報品質高
- **裝甲師全速**：6 hex/tick
- **這是進攻黃金窗口**

### Tick 5（小雨持續）的限制
- **CAS 效率 ×0.5**：3 個 sortie 只等於 1.5 個效果
- **地面開始濕滑**：移動 ×0.85
- **空中偵察 PR ×0.5**：戰場細節判讀差
- **戰術影響**：依賴 CAS 的攻勢效力被砍半

### Tick 6-7（中雨）的全面變化
- **CAS 大部分取消**：軍級僅允許緊急 Cab Rank
- **地面泥濘**：移動 ×0.7、裝甲 ×0.5
- **戰術影響**：
  - 攻方失去空中支援
  - 守方獲泥濘加成（攻方陷入）
  - 補給卡車延遲、彈藥可能短缺
  - 雙方都損失移動自由
  
- **歷史教訓**：1944 諾曼第戰役多次因天氣轉差攻勢失敗

### Tick 8（道路泥濘極端）
- **CAS 限緊急用**
- **戰車移動限 0.5 hex/hour**
- **步兵體力消耗 ×1.5**（拖步靴難拔出）
- **這是巷戰最痛苦的天氣**——攻方優勢全消失

## V. 預報能力

### 軍級氣象組

- **0-12 hour 預報**：高準度（依目視+氣壓觀測）
- **12-24 hour 預報**：中等準度（方向正確、精確度差）
- **24-48 hour 預報**：粗略（「明天可能下雨」）
- **48+ hour**：戰役級判讀（季節+海面氣旋）

**注：** 1944 年無衛星圖像，僅靠地面氣象站網+艦船觀測。盟軍有 Bletchley 解密德軍氣象電報→額外情報。

### 玩家可看到的資訊

每 Tick 開頭：
```
氣象組報告（Tick N）：
  當前：晴朗 16°C 微風 SW
  6 hr 後預測：多雲、能見極佳（HIGH 信心）
  12 hr 後預測：多雲、可能下午轉陰（MED 信心）
  24 hr 後預測：陰天可能轉雨（LOW 信心）
  
  戰術影響：
  - 接下來 12 hr CAS 條件理想
  - 24 hr 後氣壓下降，可能進入低雲區
```

戰役關鍵時刻：
```
Tick 3（夜間轉黎明前）氣象警告：
  ⚠️ 西北方氣壓系統接近
  ⚠️ 預計 06:00 起天氣轉差、CAS 效率下降
  ⚠️ 12:00 後可能轉雨
  建議：依賴 CAS 的攻擊計畫提前到 Tick 1-3 完成
```

## VI. 反季節考量

### 8 月底法國氣候特性
- 平均 23 天 / 30 天無雨
- 但有大氣旋系統穿過（1-3 次/月）每次帶來 2-4 日雨
- 海邊湧濕氣 → 晨霧頻繁（特別 Cotentin 半島）
- 戰役期間 30% 機率遇到 1 次大氣旋

### 1944 諾曼第實際統計
- D-Day 前氣象決定推遲 24 hour（Eisenhower 著名決定）
- 整 6 月 1-30 日：18 天可飛、12 天無法飛
- 7 月：類似
- 8 月：較好、但仍有雨日

**Run #1 與本戰役的天氣假設**：採用了實際 1944-08-25 至 28 的歷史記錄、加少量戰役化簡化。
