## IV. 命令引用規則

### 玩家下令時的引用檢查

當你下令包含具體小單位/裝備時，我會檢查 TOE：

```
你下令：「派 Ranger 狙擊組 3 人去 (13,14)」
我檢查：第 2 Ranger 營有 M1903A4 × 18、6 個連、戰役期間附屬到你的 III Corps
→ 通過。可派 3 人。彈藥扣 .30-06 ×12（穿甲彈+標準）+ Welrod 9mm × 6
```

```
你下令：「3AD 派 2 個 Panther 戰車營反擊」
我檢查：美軍**不裝備 Panther**——3AD 是 M4 Sherman + M5 Stuart
→ 拒絕。並提醒「你方無 Panther，是否指 M4 Sherman 兩個營？」
```

### Adler 下令時同樣檢查

```
Adler 下令：「Tiger 連對 (15,9) 反擊」
我檢查：17SS-PzGr 是 Stug III 師，**無 Tiger**
```
