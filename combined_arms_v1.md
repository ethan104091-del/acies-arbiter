# 兵種協同系統 v1（**確定性版**）

> 配合 `combat_v1.md`、`forces_v1.md`、`recon_v1.md`、`urban_combat_v1.md`。協同是 WWII 戰場勝負關鍵——本檔寫明步戰砲空工 5 兵種如何協同、如何破裂、如何計算具體倍率。**無擲骰**。

---

## 0. 設計原則

### 什麼是 Combined Arms
不同兵種「**相互保護彼此弱點**」：
- 步兵不能殺重戰車 → **戰車**殺
- 戰車怕單兵 AT → **步兵**清掩護
- 步戰怕間瞄砲擊 → **CAS** 殺敵砲位
- CAS 怕 AA → **步兵**掩護 AA 位置
- 所有兵種怕雷區 → **工兵**清

### 為何協同 = 倍率

兵種彼此**配合好**：力量倍增
兵種彼此**斷裂**：戰鬥力崩潰

**Run 1 重算**：4ID 強渡中橋因 CAS（包 C）打 AT 殺戮區 = 三兵種協同（步+空+砲 1ID）= ×1.5 倍率。**這個倍率不是擲骰**、是條件確定對應的數字。

## I. 五兵種定義

| 兵種 | 主要任務 | 弱點 |
|------|---------|------|
| **步兵** | 佔領、巷戰、清雷、近距戰 | 對戰車弱、被砲擊壓制 |
| **裝甲（戰車）** | 突破、機動、火力 | 巷戰、雷區、AT 步兵組 |
| **砲兵** | 間瞄火力、反砲擊、阻滯 | 機動慢、需 FO、彈藥消耗 |
| **空軍（CAS）** | 對地打擊、偵察、阻滯 | 天氣、AA、需 FAC |
| **工兵** | 拆雷、架橋、爆破、工事 | 戰鬥力弱、需掩護 |

## II. 協同倍率表（**確定**）

| 兵種組合 | 倍率 | 範例 |
|---------|------|------|
| 單兵種 | 1.0 | 純步兵攻擊 |
| 雙兵種（步+戰、戰+砲、步+砲、步+空、戰+空、步+工等）| 1.3 | 步戰協同（Sherman 帶步兵連）|
| 三兵種 | 1.5 | 步+戰+砲（標準攻擊配比）|
| 四兵種 | 1.7 | 步+戰+砲+空 |
| 五兵種全備 | 1.85 | 步+戰+砲+空+工（完美攻擊配套）|

**注：** 必須**實際在戰場上互相支援**才算。CAS 飛抵+步戰在地面+砲兵 FO 同時在場才有效。光「軍級擁有」不算。

## III. 各兩兵種協同的細節

### 1. 步兵 + 裝甲（最常見）

**戰術組合：**
- 1 個戰車排 + 1 個步兵連
- 戰車提供火力+反工事破壞
- 步兵清掩護戰車後/側、提供 AT 警戒

**互補效應：**
- 戰車殺強點（碉堡、據點）→ 步兵不需自己衝
- 步兵清近距 AT 威脅（Panzerfaust 組）→ 戰車不損失
- 戰車的 .50 cal MG 壓制窗口 → 步兵衝建築物

**範例：3AD CCA 攻擊**
- 3AD CCA = 1 個戰車營（54 輛 M4）+ 1 個裝甲步兵營（M3 半履帶）
- 標準戰術隊形：戰車 + 半履帶步兵推進
- 協同倍率：×1.3（雙兵種）

**斷裂條件：**
- 戰車前出超過步兵 200m → 失去掩護
- 步兵跟不上速度 → 戰車孤立
- 通信中斷 → 不能協同

### 2. 步兵 + 砲兵

**戰術組合：**
- 步兵團 + 師屬砲兵營（105mm × 12）
- 步兵 FO 跟隨第一線連
- 砲兵 FDC 在團/師後方

**互補效應：**
- 步兵 FO 觀察 → 即時呼叫砲擊
- 砲兵壓制敵步兵集結
- 反砲擊摧毀敵砲兵

**範例：1ID 攻擊 Saint-Vivien 西側**
- 1ID 第 18 步兵團 + 第 7 砲兵營（M2A1 105mm × 12）
- FO 在團指揮所 + 隨第一線連推進
- TOT（Time-on-Target）打擊：4-7 分鐘抵達
- 協同倍率：×1.3

**斷裂條件：**
- FO 陣亡 → 砲兵失效
- FO 通信中斷 → 砲擊延遲 30-60 分
- 砲彈短缺 → 火力打折

### 3. 裝甲 + 砲兵

**戰術組合：**
- 戰車師 + 軍級附屬砲兵營
- 戰車進攻+砲兵壓制守方 AT

**互補效應：**
- 砲兵壓 AT 陣地 → 戰車可前進
- 戰車突破 → 砲兵跟著拉到新位置（自走砲特別有效）
- 美軍 M7 Priest 自走砲（105mm SP）= 戰車師標配

**範例：3AD 突破 La Hêtraie**
- 3AD 戰車 + 第 54 自走砲營（M7 Priest × 18）
- 砲兵打 La Hêtraie 守軍工事 → 戰車突入
- 協同倍率：×1.3

### 4. 步兵 + 空軍

**戰術組合：**
- 步兵 + FAC（Forward Air Controller）
- 步兵呼 P-47 從天而降

**互補效應：**
- FAC 指揮 P-47 打殺戮區、防禦點
- 步兵推進、CAS 清前方威脅

**範例：4ID 強渡中橋 + CAS 包 C 打 AT 殺戮區**
- 4ID FAC 配軍級 ALO
- CAS 包 C 打 (15, 9-11) AT 砲位
- 4ID 在 CAS 之後 90 分鐘衝橋
- 協同倍率：×1.3

**斷裂條件：**
- FAC 設備故障 → CAS 失準
- 天氣阻 CAS → 不能協同
- AA 高 → CAS 損失過大、效率下降

### 5. 裝甲 + 空軍

**戰術組合：**
- 戰車主力 + 上空 P-47 待命

**互補效應：**
- P-47 殺敵 AT 砲位 → 戰車可推
- P-47 偵察 → 戰車避開埋伏
- P-47 殺敵戰車（頂裝甲）→ 戰車補救

**範例：3AD 過北橋後配 P-47 Cab Rank**
- Cab Rank P-47 在頭上盤旋
- 戰車遇 AT 威脅立呼
- 反應時間 5-15 分鐘

### 6. 步兵 + 工兵（**極關鍵但常被忽略**）

**戰術組合：**
- 步兵連 + 1-2 個工兵班

**互補效應：**
- 工兵清雷 → 步兵可推進
- 工兵爆破門/牆 → 步兵突入
- 步兵掩護工兵作業 → 工兵不被狙

**範例：90ID 工兵拆南橋預埋（Run 1 Tick 3）**
- 90ID 第 315 工兵戰鬥營
- + Ranger 連狙擊組（軍級附屬）
- 步戰特戰協同 = ×1.3 但 Ranger 是特殊狙擊兵種

### 7. 工兵 + 砲兵（極少配但有效）

- 工兵預埋爆破 + 砲兵掩護
- 用於防禦準備（如 Adler 預埋中橋爆破）

### 8. 工兵 + 戰車

- 工兵清雷 + 戰車隨進
- 美軍 Sherman dozer（推土機推前面）
- 工兵架橋 + 戰車過

## V. 砲兵 FO/FDC 鏈條（詳細）

### 砲擊呼叫流程（**確定**時序）

```
T+0 sec: FO 看到目標、決定打
T+10 sec: FO 用 SCR-300 步話機呼 → 團 FDC
T+30 sec: 團 FDC 用 SCR-193 中功率呼 → 師 FDC
T+1 min: 師 FDC 計算座標、決定砲組
T+2 min: 師 FDC 命令到砲位
T+3 min: 砲組瞄準
T+4 min: 開火
T+4.5 min: 第一彈著
T+5 min: FO 觀察彈著、計算修正
T+6 min: 修正命令傳回
T+7 min: 第二彈著（精確）

** 標準延遲：4-7 分鐘 **

** 美軍 TOT 加速版：
- 多砲組同時瞄同一目標
- 計算飛行時間後同時開火
- 多發同時抵達目標 = 巨大震撼效果
- 但籌備 5-10 分鐘
```

### FO 配置

| 單位 | FO 數量 |
|------|--------|
| 美軍步兵團 | 3-4 個 FO 組（每營 1 + 團本部）|
| 美軍裝甲師 | 每戰車營 2-3 個 FO 組 |
| 德軍步兵團 | 2-3 個 FO 組（少於美軍）|

### FO 陣亡後果

- 該團砲兵呼叫延遲到 30-60 分鐘
- 砲兵效力 ×0.5
- 24-48 小時補新 FO

### Counter-Battery（反砲擊）詳細

```
條件：對方砲兵開火、暴露位置

1. 你方 DF 設備（SCR-625）測向 → 鎖定大致方位
2. 觀察員觀察（彈著閃光、煙、聲音）→ 確認具體座標
3. 反砲擊命令 → 軍級 / 集團軍砲兵
4. 反砲擊執行 → 5-15 分後對該位置射擊

對方應對：
- 打了就走（M7 自走砲、機動性高）→ 反砲擊失敗
- 拖砲（leFH 18）：移動需 30-45 分 → 反砲擊可能擊毀
- 隱蔽位置（hull-down、樹蔭）：反砲擊精度下降
```

## VI. CAS-FAC-ALO 鏈條（詳細）

### CAS 呼叫流程

```
T+0 min: 步兵連長看到目標、需 CAS
T+1 min: 連長呼營 FAC
T+2 min: 營 FAC 評估、呼軍級 ALO
T+3 min: ALO 確認任務、呼軍級航空指揮部
T+5 min: 航空指揮部派 Cab Rank（盤旋中）P-47
T+10 min: P-47 抵達目標上空
T+15 min: P-47 攻擊執行

** Cab Rank 標準延遲：5-15 分鐘 **

** 非 Cab Rank（新請求）：
- 6 小時前置時間
- 不能即時呼叫
- 適合預定打擊（如 Run 1 Tick 4 黎明 CAS 預訂）
```

### FAC（Forward Air Controller）

**美軍 FAC 編組：**
- 軍級 ALO（Air Liaison Officer）+ 4 個師級 FAC 小組
- FAC 小組：2-3 人（軍官 + 通信員 + Jeep）
- 通信：SCR-522 V/UHF 對機+地對地

**FAC 任務：**
- 即時目標識別
- 引導 P-47 攻擊位置
- 評估 CAS 效果
- 為砲兵提供地對空通信

### FAC 損失後果

- 該師失去 CAS 即時呼叫能力
- 預訂 CAS 仍可（軍級 ALO 處理）
- 24-48 小時補新 FAC

## VII. 兵種協同的破裂條件（**確定**）

### 何時協同失效

| 破裂條件 | 後果 |
|---------|------|
| 通信中斷（無線電被干擾/設備故障）| 對應兵種倍率歸 1.0（剩單兵種）|
| 該兵種短缺（彈藥/燃料 <10%）| 該兵種視同不在場 |
| 該兵種疲勞 >80 | 倍率 ×0.7 |
| FO/FAC 陣亡 | 該兵種協同 ×0.5 直到補 |
| 天氣阻空中 | 空軍不算協同 |
| 補給線斷 | 砲兵彈藥可能跌、CAS 不受影響 |

### 範例：Run 1 Tick 5 4ID 強渡中橋

```
協同檢查：
✓ 步兵：4ID 主力
✓ 戰：4ID 沒有專屬戰車營但與 1ID 戰車營掩護
✓ 砲：1ID 砲營 + 4ID 砲營
✓ 空：CAS 包 C
✓ 工：4ID 工兵營（搭浮橋）
=> 五兵種全備、協同倍率 ×1.85

但 4ID 自身組織度只剩 46（中度疲勞）：
+ 接戰時又被打到 25 → 接近崩潰
→ 整體 CP 仍低、即使 ×1.85 也不能勝過守方 145 CP

Run 1 我用 ×1.5（三兵種）算保守、正確值應是 ×1.85
公式版顯示協同倍率更高、但守方城戰倍率也大
→ Force Ratio 變化在 0.17-0.25 之間
→ 結論仍然是「攻方失敗、傷亡慘」**正確**
```

## VIII. 兵種協同的歷史教訓

### 美軍 vs 德軍協同能力對比（1944）

| 維度 | 美軍 | 德軍 |
|------|------|------|
| 步戰協同 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐（傳統優勢）|
| 步空協同（CAS）| ⭐⭐⭐⭐⭐ | ⭐⭐（西線 1944 已弱）|
| 步砲協同 | ⭐⭐⭐⭐⭐（FO 系統好）| ⭐⭐⭐⭐ |
| 戰砲協同 | ⭐⭐⭐⭐（自走砲完整）| ⭐⭐⭐ |
| 步工協同 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐（東線經驗）|

### 戰役級協同差異

美軍 1944 西線勝利的關鍵：
- **CAS 體系完整**：每個師有 FAC、Cab Rank、ALO
- **砲兵 FO 標準化**：每團每營都有
- **空優壓制下協同自由**：天空安全、地面協同可全力
- **無線電量充足**：通信不會斷

德軍 1944 西線困境：
- **無 CAS**：Luftwaffe 已退出西線
- **協同破裂**：缺通信、缺燃料、缺彈藥
- **單兵種戰力強**（如 Tiger）但孤立易被殲

## IX. 玩家如何運用協同（戰術選擇）

### 戰術 1：集中協同攻擊（高倍率）

**情境：** 你要攻 Saint-Vivien 主城門

**選擇：** 全套協同
- 1ID 步兵 + 1ID 戰車營
- 1ID 砲營 + 軍級 4.2" 化學迫擊（軍級資源）
- CAS 2 個 sortie 預訂
- 1ID 工兵清雷

**結果：** ×1.85 倍率 + 雷區清+ AT 殺戮區先打掉

**代價：** 物資消耗大、需 24 小時準備、CAS 用掉

### 戰術 2：先協同後孤軍（混搭）

**情境：** 主軸協同攻、次軸孤軍偵察

**選擇：**
- 主軸 1ID 全協同攻
- 次軸 3AD 偵察排+輕戰車獨進（×1.0）

**結果：** 主軸效率高、次軸彈性大但慢

### 戰術 3：協同破裂的代價

**情境：** Run 1 Tick 8 你打 Saint-Vivien 但 CAS 包 C 已用

**結果：** 失去空中協同、整體倍率從 ×1.85 降至 ×1.5
→ 4ID 的攻擊力嚴重縮水

## X. 完整協同範例：Saint-Vivien Tick 8 重算

### 4 個師同時進攻 Saint-Vivien

```
1ID 主攻 (17,9)-(18,9) 西側：
  步：1ID 三個團
  戰：1ID attached 745 戰車營
  砲：1ID 砲營 + 軍級 4.2" 迫擊（87 Chem Bn）
  空：CAS 包 A（HQ 斬首延遲信管）
  工：1ID 工兵營
  → 五兵種 ×1.85

4ID 跟進攻 (17,10)：
  步：4ID
  戰：4ID 70 戰車營
  砲：4ID 砲兵 + 共用 1ID 砲兵
  空：CAS Cab Rank
  → 四兵種 ×1.7

3AD 北攻 (18,7)：
  步：3AD 裝甲步兵團
  戰：3AD 兩個戰車營
  砲：3AD 自走砲營（M7 Priest × 54）
  空：CAS 包 B
  工：3AD 工兵營
  → 五兵種 ×1.85

90ID 南攻 (19,11)：
  步：90ID 步兵團
  戰：90ID 712 戰車營（但戰力低）
  砲：90ID 砲營
  空：無（Cab Rank 已用完）
  → 三兵種 ×1.5

協同合計效應：
  4 軸同時攻擊
  Saint-Vivien 城內守軍 4 個方向都要應對
  即使各軸 Force Ratio 都接近 1.0、4 軸同時 = 守方分兵
  守方一定有 1 軸被突破
```
