# 偵察與情報系統 v1（**確定性版**）

> 配合 `determinism_v1.md`。**核心原則：無機率、無擲骰。** 偵察結果由「對方的具體部署狀態」與「我方的偵察手段能力」**確定**對應。
>
> 不確定性的來源是：**對方做了什麼你不知道**，而不是「擲骰看你今天運氣」。

---

## 0. 設計原則（讀者必看）

### 不擲骰，怎麼維持戰場不確定性？

三個機制取代機率：

1. **狀態驅動**：每個單位有明確「能見狀態」（EXPOSED / STANDARD / CAMOUFLAGED / CONCEALED / HIDDEN）。狀態決定**確定**的可觀察性。
2. **資訊不對稱**：你能看到自己部隊狀態、敵方部隊只能透過偵察觀察。對方做了什麼選擇你不知道——這就是不確定性。
3. **徵候判讀**：觀察手段看到的是**徵候**（電話線、車轍、煙），不是「結果」。徵候對應**多種可能解釋**，你必須推理。

### 為什麼這比機率更真實

- 真實戰場：F-6 飛過 (15,9) 不會「擲骰決定看不看到 88」。它**確實看到**地面狀態。地面狀態是「88 在那 + 偽裝完整 + 樹蔭」就拍出**那個樣子的照片**。
- 真實戰場：PI 不是「擲骰判讀」。它**確實能**或**確實不能**從照片識別。判讀技術+影像品質決定。
- 真實戰場：你不知道 Adler 用了多少帆布偽裝——這個資訊你**沒有**，跟「擲骰」根本兩件事。

## I. 單位能見狀態（5 級）

每個單位在每個時刻有一個能見狀態。**狀態由部署決定**，玩家/Adler 寫明。

### EXPOSED — 暴露
- 開闊地、無偽裝、移動中、白天
- 條件：在平原 hex 且 30 分內有過移動 OR 開砲 OR 引擎運轉
- 例：縱隊行軍中的戰車

### STANDARD — 標準部署
- 預設狀態（無偽裝行動但未暴露）
- 條件：靜止 + 部分自然遮蔽（建築旁、坡背、灌木）或部分偽裝（樹葉手工蓋）
- 部隊「進駐位置但未做隱蔽工作」的狀態
- 例：剛抵新陣地的步兵營

### CAMOUFLAGED — 偽裝
- 軍方標準偽裝完成（網、塗、樹葉組合）
- 條件之一：
  - 投入帆布偽裝網 **50 yd²/輛 + 樹蔭** OR
  - 投入帆布偽裝網 **100 yd²/輛** 純偽裝
  - 投入 4 小時人力偽裝工作
  - 戰車進入掩體（hull-down）+ 額外偽裝
- 例：完成偽裝的 88mm AT 砲位

### CONCEALED — 隱蔽
- 高度遮蔽 + 完整偽裝
- 條件全部達成：
  - **150 yd² 帆布偽裝網/輛**
  - **森林深處** OR **建築物內**
  - **2 小時靜止無動作**
  - **無線電靜默或備頻**
- 例：Run 1 你 3AD 藏 (9,3)F 北森林

### HIDDEN — 完全隱蔽
- 隱蔽+全套抗偵察
- 條件全部達成：
  - 達 CONCEALED
  - **超出敵方所有偵察手段的範圍**
  - **連續 6 小時無任何動作**（不開火不移動不發報）
  - **車輛已熄火 1 小時以上**（無紅外熱徵候——本戰役無紅外但未來擴充）
- 例：你 1ID 在 (4,4) 西側深處——超出 Adler 偵察可達範圍

### DECOY — 假目標
- 不是單位本身的狀態，是**部署的假象**
- 由空帆布+木材+少量人員製造「看似有單位」
- 條件：
  - **200 yd² 帆布** + **木材 100 bf** + **6 人 4 小時工作**
- 例：Run 1 Adler 在 (15,11) 設的假 88

## II. 偵察手段對應狀態的**確定**觀察結果

下表是**確定關係**。同樣狀態同樣手段同樣結果。**沒有擲骰**。

### F-6 Mustang TacR（低空 1,500-3,000 ft，斜角+垂直相機）

| 目標狀態 | 觀察結果 |
|---------|---------|
| EXPOSED | **IDENTIFIED**：可識別兵種、估計數量、座標精確至 0.5 hex |
| STANDARD | **IDENTIFIED**：兵種+數量可識別（精度略降）、座標 1 hex 內 |
| CAMOUFLAGED | **DETECTED**：「有東西在那」、不能精確識別兵種、座標 2 hex |
| CONCEALED | **MISSED**（**確實看不到**）|
| HIDDEN | **MISSED**（甚至看不到該區域的線索）|
| DECOY | **IDENTIFIED**：看到對方「想讓你看到的」、與真實 STANDARD 部隊**外觀完全相同**——你只能透過**徵候完整性**判別 |

### F-5 Lightning PR（高空 20,000 ft，多相機立體對）

| 目標狀態 | 觀察結果 |

## III. 徵候判讀（資訊整合）

偵察手段看到的是**徵候**，徵候本身**確定可見**。但徵候是否代表 X 是**推理**問題。

### 範例：師級指揮所典型徵候清單

師級指揮所**會產生**以下徵候（不可避免——一個師需要這些設施運作）：

| 徵候 | 觀察方式 | 是否必然存在 |
|------|---------|------------|
| 多條野戰電話線匯集 | TacR/PR | ✅ 師指必有 |
| 無線電天線陣列（多支高桿）| PR/L-5 | ✅ 師指必有 |
| 車輛集中（Kübelwagen+SdKfz 251 等指揮車）| TacR/PR | ✅ |
| 摩托傳令兵頻繁出入 | TacR（瞬間捕捉）/ L-5（持續觀察）| ✅ |
| 屋頂觀察哨 / 觀察平台 | PR（立體判讀）| ✅（大多）|
| 防空陣地集中（保護要員）| TacR/PR | ✅ |
| 補給車隊優先卸貨點 | TacR | ✅ |
| **缺乏戰鬥部隊集中**（HQ 不是部隊集結點）| PR 立體分析 | ✅ |

**判讀邏輯：**

```
F-6 TacR 拍到 (18,9) 市政廳：
  ✅ 多條電話線匯集 → ✅
  ✅ 3 輛 Kübelwagen 集中 → ✅
  ✅ 摩托傳令兵活動 → ✅
  ✅ 防空陣地（20mm Flak × 2）→ ✅
  ⚠️ 無法觀察：屋頂觀察哨（圖像角度問題）、地下化（不可見）

判讀結論：(18,9) 為 17SS 師級指揮所
信心：HIGH（5/7 徵候已確認，缺 2 但不是必要徵候）
```

**徵候完整性決定判讀信心，不擲骰**：
- **CONF（確認）**：5+ 徵候全到位
- **HIGH（高）**：3-4 個徵候到位
- **MED（中）**：2 個徵候 + 邏輯支持
- **LOW（低）**：1 個徵候 + 推測
- **SPEC（推測）**：無直接徵候、僅推理

### 假目標如何識破

DECOY 部署會缺少某些徵候——這就是辨假的方法：

```
假 88 砲位需要 200 yd² 帆布 + 木材 100 bf + 6 人 4 小時工作
但它缺少：
  - 砲手宿舍/休息點
  - 彈藥堆積（真的 88 需要彈藥車進出）
  - 觀察員位置/前進通信線
  - 進出車轍痕跡

PR 立體判讀：
  ✅ 砲管形狀正確（帆布做的也可像砲管）
  ❌ 無彈藥堆積支援設施
  ❌ 無進出車轍
  → 判讀：「可能為假目標」MED 信心

對策（Adler 增加假目標真實性）：
  - 加木材做「假彈藥箱」
  - 派 4 人定期進出做車轍
  - 加偽裝營房
  - 成本：×2 帆布+木材+人力
  → 升級為「難以分辨」DECOY
```

## IV. 玩家如何具體決策（不靠機率）

### 範例 1：你想隱藏 3AD

**舊版（機率）：** 「我藏 3AD 進森林，希望 Adler 找不到（70%）」

**新版（狀態）：**
```
你下令：「3AD 移至 (9,3)F、達 CONCEALED 狀態」
你的物資扣：
  - 帆布偽裝網：3AD 戰車 92 輛 × 50 yd² = 4,600 yd²
  - 你 3AD 物資池目前帆布庫存：~ 20,000 yd²（裝甲師高量）
  - 扣 4,600 → 剩 15,400
  - 樹蔭：(9,3)F 為森林 hex，免費取得
  - 無線電靜默：你的命令包含

→ 3AD 確認進入 CONCEALED 狀態
  - F-6 TacR 飛過：MISSED
  - F-5 PR 飛過：MISSED（但可能 INDICATOR）
  - 地面偵察隊接觸 1 hex 內：DETECTED
  - SIGINT：MISSED（無發報）
```

不再「我 70% 藏住」。是「我具體做了 X、Y、Z 動作、狀態確定為 CONCEALED、各種偵察手段的結果為 ABC」。

### 範例 2：你想識破 (15,11) 是假 88

```
你下令：「派 L-5 砲兵觀測機在 (14,11) 上空盤旋 30 分」
扣 sortie：L-5 配額有 ~6 個/天可調
結果：
  L-5 對 (15,11) 連續 30 分鐘觀察：
  - 觀察期間是否有「彈藥車輛進出」？ ← 你問
  - 觀察期間是否有「砲手活動換班」？ ← 你問

L-5 報告（無延遲、即時無線電）：
  - 30 分鐘內 (15,11) 無任何車輛/人員活動
  - 砲管位置完全靜止，無調整跡象
  → 與真實 88 砲位該有的活動水平不符
  → 判讀「(15,11) 為假目標」MED-HIGH 信心
```

你**用具體資源換取具體情報**。沒有骰子。

### 範例 3：你想知道 (18,9) 是不是 HQ

```
方法 1：PR 一次任務
  → 看到 4-5 個徵候、HIGH 信心是 HQ

方法 2：方法 1 + SIGINT 監聽
  → 截聽到師長呼號從 (18,9) 區域發報、確認

方法 3：方法 1 + 戰俘審訊（如果有戰俘）
  → 戰俘確認師長在 (18,9)、CONF 信心

方法 4：派 Ranger 偵察組接觸（高風險）
  → 直接目視，但可能戰鬥損失

每種方法有成本（sortie、戰俘來源、風險），給你**確定的情報加成**。
```

### 範例 4：Adler 欺敵的可分辨性

```
Adler 在 (16,11) 設假砲位：
  - 投入 200 yd² 帆布、木材 100 bf、6 人 4 小時
  - **如果他這就完了**：你 PR 一次可能識破（缺支援徵候）
  - **如果他加碼**：加假彈藥箱 + 派人定期車轍 → 提升到「難分辨」級別
  
他的選擇影響你判讀的難度——不是擲骰。
```

## V. 「徵候完整性」對應到信心等級

我每次答情報問題用這個邏輯：

```
你問：「Saint-Vivien 城內有多少德軍？」

我答：
  來源：F-6 TacR 1944-08-26 18:30 + F-5 PR 上一日 + SIGINT 持續
  
  在 (17-20, 9-10) 區域觀察到：
  ✅ 主要街區有摩化部隊集結（SdKfz 251 × 12+）— TacR CONF
  ✅ 防禦工事構築中（戰壕、街壘）— PR CONF
  ✅ 砲兵 6 門位置定位 — PR CONF + counter-battery DF
  ✅ 通信中心徵候 — TacR HIGH
  ⚠️ 戰車數量不確定 — 城內樹蔭+建築掩護、僅看到部分
     觀察數：~6 輛戰車（看到的）
     **不可觀察**：建築物之間隱蔽的戰車（估計 +5-10 輛 LOW）
  ⚠️ 步兵總數不確定 — 城內人員散佈廣
     觀察推估：1 個團規模（2,500-3,500 人）MED
     可能誤差：±20%
  ❌ 地下化部分完全不可觀察

綜合：
  17SS 主力在 Saint-Vivien 確認 HIGH
  戰車估計 6-16 輛 MED
  步兵估計 2,500-3,500 MED
  指揮所在 (18,9) 機率高 HIGH（從 III 章分析）
  
信心整體：MED-HIGH
```

每條情報都有**確定的觀察基礎**+**明確的未知部分**。

## VI. 反偵察成本（確定，不擲骰）

| 動作 | 物資消耗 | 效果 |
|------|---------|------|
| 1 輛戰車升 EXPOSED → CAMOUFLAGED | 帆布 50 yd² + 樹蔭 OR 100 yd² 純 | 確定 CAMOUFLAGED |
| 1 輛戰車升 → CONCEALED | 加 50 yd² + 森林深處 + 2h 工作 + 無線電靜默 | 確定 CONCEALED |
| 1 個營升 → CAMOUFLAGED | 200 yd² 帆布/連 × 3 連 = 600 yd² | 確定 CAMOUFLAGED |
| 1 個師全升 → CAMOUFLAGED | 視兵種需 8,000-15,000 yd² 帆布 | 確定 |
| 假目標營級規模 | 帆布 1,000 yd² + 木材 400 bf + 24 人 6 小時 | DECOY 部署 |
| 無線電靜默（單位）| 戰術代價：失去師-團聯絡 | SIGINT 完全 MISSED |
| 換頻率 | 信號連協調工作 1 小時 | SIGINT 效力 -40%（持續期間）|

物資不足 = 不能達到該狀態。**確定的因果**。

## VIII. 不擲骰但仍保留戰爭迷霧

關鍵：**你（玩家）看不到 Adler 的選擇**——

- 你不知道他用了多少帆布
- 你不知道他偽裝了哪些單位
- 你不知道哪個是假目標
- 你不知道他的徵候完整度

當你 TacR 飛過：**我輸出你能看到的徵候**（依他選擇+你方法的確定關係）。
你**推理**那些徵候代表什麼——這就是真實情報軍官的工作。

對方也一樣不知道你的選擇——這是**對稱**的資訊不對稱。

骰子完全不在這個系統裡。
